# bose-landing
Clean and modern product landing page showcasing Bose audio devices. Built with semantic HTML, CSS Grid, and Flexbox.

## Project Description

This project is a static landing page that presents Bose audio products.  
The website includes:

- Full-screen header section
- Slide-out navigation menu
- Recommended products section
- Product categories section
- Benefits section
- Sticky footer with background image

The layout is built using Flexbox and CSS Grid.

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Google Fonts (Inter)

## Features

- Smooth scroll navigation
- Slide-out menu using :target
- Responsive grid layout
- Hover effects
- Clean and modern UI
